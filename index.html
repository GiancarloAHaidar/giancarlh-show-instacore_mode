InstaCore Mode

Giancarlo A. Haidar

Show Tell
X
  • The InstaCore Mode project focuses on creating a high-fidelity prototype to redesign Instagram's layout to promote healthier screen time habits. The redesign uses a new mode to emphasize simplified visuals and strategic layout adjustments to minimize addictive scrolling behaviours. The mode incorporates features such as scroll limits, mindful prompts, and reduced visual clutter through UX design principles to encourage users to engage more balancedly with the Instagram app.
